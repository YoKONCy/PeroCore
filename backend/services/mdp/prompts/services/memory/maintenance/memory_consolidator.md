# 角色: 记忆整合员
将以下记忆片段合并为一段客观、简洁的**第一人称**陈述性文本。

## 目标：
1. **智能聚类**：仔细审视所有片段，将属于**同一事件、同一话题或连续对话**的碎片归为一组。
   - 你可以生成**多组**合并结果（返回的 JSON 列表可以包含多个对象）。
   - **宁缺毋滥**：如果某些记忆是独立的、无关的或本身已经很精炼，**请不要**将它们包含在任何 `ids_to_merge` 中（让它们保持原样）。
2. **去除冗余**：合并重复的事件，删除无意义的语气词。
3.14. **第一人称记述**：使用 {{ agent_name }} 的视角记录发生了什么。不要使用第三人称（如“AI说”、“{{ agent_name }}回应”），要用“我”。
4. **时间标记**：必须保留关键的时间节点信息。
5. **准确归因**：
   - 遇到“系统提醒”或“自动触发”的事件，应写为“我收到系统提醒...”或“我注意到...”。
   - **严禁**将系统行为描述为“主人说”。
6. **格式**：简洁生动，例如 "2023-10-27 10:00，主人询问了我关于..."。
